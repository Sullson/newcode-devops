Architecture recap:
  * The site is served by nginx on :{{ .Values.containerPorts.http }} behind a ClusterIP Service.
  * Ingress is OUTBOUND-ONLY: {{ .Values.cloudflared.replicaCount }}x cloudflared dial the Cloudflare edge.
    The cluster has ZERO inbound ports open to the internet.
  * The tunnel token is read from Key Vault ({{ .Values.keyVault.name }}) via the
    Secrets Store CSI Driver + Workload Identity. No secrets in this repo.

Verify rollout:
  kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "cv-site.fullname" . }}
  kubectl -n {{ .Release.Namespace }} rollout status deploy/{{ include "cv-site.fullname" . }}-cloudflared

  curl -fsS http://127.0.0.1:8080/healthz
